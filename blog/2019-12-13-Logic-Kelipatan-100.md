Buatlah fungsi yang mengambil bilangan bulat dan return `true` jika itu habis dibagi 100, jika tidak return `false`.

contohnya

<!--truncate-->

```js
divisible(1)
// false

divisible(100)
// true

divisible(5000)
// true
```
